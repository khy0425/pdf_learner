# PDF 학습기 (PDF Learner v2)

AI 기반의 PDF 관리 및 학습 플랫폼으로, 문서 읽기를 넘어 지식 습득과 관리를 혁신하는 크로스 플랫폼 애플리케이션입니다. Flutter 기반으로 개발되어 Android, iOS, Windows, macOS, Web에서 사용 가능합니다.

## 프로젝트 비전

단순한 PDF 뷰어가 아닌, AI 기술을 활용하여 사용자의 학습과 지식 관리를 혁신적으로 지원하는 지능형 플랫폼을 구축합니다. 월 100만명 이상의 사용자에게 서비스를 제공하는 것을 목표로 합니다.

## 페이지 주소
https://pdf-learner.web.app/

## 프로젝트 개요

- **프레임워크**: Flutter
- **아키텍처**: MVVM (Model-View-ViewModel)
- **상태 관리**: Provider
- **AI 통합**: Google AI
- **데이터 저장**: Firebase, 로컬 스토리지
- **분석**: Firebase Analytics, 자체 학습 데이터 분석

## 핵심 기능

### 1. AI 기반 문서 상호작용
- **PDF와 채팅**: 문서 내용에 대해 질문하고 답변 받기
- **마인드맵 자동 생성**: PDF 내용을 시각적 마인드맵으로 변환
- **지능형 요약**: 다양한 길이와 스타일의 문서 요약 생성
- **AI 기반 퀴즈 생성**: 객관식, 주관식, 단답형 등 다양한 형식 지원
- **이미지 분석**: PDF 내 이미지 인식 및 설명 생성

### 2. 고급 PDF 관리
- **문서 편집**: 텍스트 편집, 이미지 삽입/편집, 페이지 관리
- **주석 기능**: 하이라이트, 메모, 그리기, 음성 메모
- **문서 변환**: PDF ⟷ Word/Excel/PPT/이미지/텍스트
- **전자 서명**: 디지털 서명 생성 및 관리
- **OCR 기능**: 스캔된 문서 텍스트 인식 (28개 언어 지원)

### 3. 학습 최적화 기능
- **학습 진도 추적**: 읽기 시간, 페이지 진행도, 복습 패턴 분석
- **지능형 학습 계획**: AI가 분석한 학습 패턴 기반 맞춤형 학습 계획
- **플래시카드 시스템**: 스마트 복습 알고리즘 기반 기억 최적화
- **협업 학습**: 그룹 주석, 실시간 문서 공유, 학습 그룹 관리
- **학습 통계**: 상세한 학습 데이터 시각화 및 분석

### 4. 사용자 경험
- **다크 모드**: 눈의 피로를 줄이는 다크 모드 및 읽기 모드
- **음성 지원**: TTS(텍스트 음성 변환) 및 음성 명령
- **멀티 디바이스 동기화**: 모든 기기에서 원활한 이용 경험
- **맞춤 설정**: 사용자화 가능한 UI, 단축키, 제스처

## 크로스 플랫폼 전략

PDF 학습기는 웹 버전과 애플리케이션 버전을 모두 지원하는 진정한 크로스 플랫폼 서비스를 제공합니다. 이를 통해 다음과 같은 장점을 실현합니다:

### 플랫폼 포괄적 접근
- **웹 플랫폼**: 즉각적인 접근성, 설치 없이 바로 사용, 모든 브라우저 지원
- **네이티브 앱**: Android, iOS, Windows, macOS에서 최적화된 성능 제공

### 기능 차별화 전략
- **웹 버전 특화 기능**: 빠른 문서 보기, 간편한 공유, 실시간 협업, 클라우드 중심 저장
- **앱 특화 기능**: 고급 편집, 오프라인 작업, 심층 AI 분석, 시스템 통합 기능

### 동기화 및 연속성
- **크로스 플랫폼 동기화**: 한 플랫폼에서 시작한 작업을 다른 플랫폼에서 이어서 진행
- **일관된 사용자 경험**: 모든 플랫폼에서 통일된 디자인 언어와 사용자 인터페이스

### 기술적 최적화
- **웹 최적화**: 웹 워커 활용, 점진적 로딩, 클라우드 처리 활용
- **앱 최적화**: 네이티브 API 활용, 하드웨어 가속, 로컬 저장소 최적화

### 비즈니스 이점
- **시장 확대**: 다양한 플랫폼의 사용자층 포괄
- **사용자 유지율 향상**: 다양한 상황에서의 접근성 제공으로 활성 사용자 증가
- **다양한 수익 채널**: 웹 구독과 앱 내 결제의 이중 수익 구조

경쟁사와 달리 PDF 학습기는 어떤 디바이스에서든 최적화된 경험을 제공하는 데 중점을 두며, 사용자가 플랫폼에 종속되지 않고 자유롭게 작업할 수 있는 환경을 구축합니다.

## 개발 환경 설정

1. Flutter 설치
```bash
git clone https://github.com/flutter/flutter.git
export PATH="$PATH:`pwd`/flutter/bin"
flutter doctor
```

2. 프로젝트 클론
```bash
git clone https://github.com/khy0425/pdf_learner.git
cd pdf_learner
```

3. 의존성 설치
```bash
flutter pub get
```

4. 환경 변수 설정
```bash
cp .env.example .env
# .env 파일을 수정하여 API 키 등 설정
```

5. 애플리케이션 실행
```bash
flutter run
```

## 공정 사용 정책 (Fair Usage Policy)

PDF 학습기는 모든 사용자에게 최상의 서비스를 제공하기 위해 다음과 같은 공정 사용 정책을 적용합니다:

### 일반 원칙
- 모든 플랜은 개인 학습 및 문서 관리 목적으로 설계되었습니다
- 상업적 대량 처리, 자동화된 스크립트를 통한 API 접근 등은 허용되지 않습니다
- 서비스 남용 시 일시적으로 기능이 제한될 수 있습니다

### 프리미엄 플랜 사용량 정책
- 표시된 일일/월간 한도는 대부분의 사용자에게 충분한 수준으로 설정되었습니다
- 정상적인 학습 목적인 경우, 제한에 도달하더라도 추가 요청이 가능할 수 있습니다
- 단, 과도한 사용량이 감지될 경우 시스템이 일시적으로 요청을 제한할 수 있습니다
- 특별히 많은 사용량이 필요한 교육기관은 팀/교육 플랜으로 전환을 권장합니다

PDF 학습기는 사용 패턴을 분석하여 서비스 품질을 지속적으로 개선합니다. 비정상적인 사용 패턴이 감지될 경우 해당 계정에 대한 검토가 이루어질 수 있습니다
