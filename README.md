# PDF 학습기 (PDF Learner v2)

AI 기반의 PDF 관리 및 학습 플랫폼으로, 문서 읽기를 넘어 지식 습득과 관리를 혁신하는 크로스 플랫폼 애플리케이션입니다. Flutter 기반으로 개발되어 Android, iOS, Windows, macOS, Web에서 사용 가능합니다.

## 프로젝트 비전

단순한 PDF 뷰어가 아닌, AI 기술을 활용하여 사용자의 학습과 지식 관리를 혁신적으로 지원하는 지능형 플랫폼을 구축합니다. 월 100만명 이상의 사용자에게 서비스를 제공하는 것을 목표로 합니다.

## 프로젝트 개요

- **프레임워크**: Flutter
- **아키텍처**: MVVM (Model-View-ViewModel)
- **상태 관리**: Provider
- **AI 통합**: OpenAI, Google AI
- **데이터 저장**: Firebase, 로컬 스토리지
- **분석**: Firebase Analytics, 자체 학습 데이터 분석

## 핵심 기능

### 1. AI 기반 문서 상호작용
- **PDF와 채팅**: 문서 내용에 대해 질문하고 답변 받기
- **마인드맵 자동 생성**: PDF 내용을 시각적 마인드맵으로 변환
- **지능형 요약**: 다양한 길이와 스타일의 문서 요약 생성
- **AI 기반 퀴즈 생성**: 객관식, 주관식, 단답형 등 다양한 형식 지원
- **이미지 분석**: PDF 내 이미지 인식 및 설명 생성

### 2. 고급 PDF 관리
- **문서 편집**: 텍스트 편집, 이미지 삽입/편집, 페이지 관리
- **주석 기능**: 하이라이트, 메모, 그리기, 음성 메모
- **문서 변환**: PDF ⟷ Word/Excel/PPT/이미지/텍스트
- **전자 서명**: 디지털 서명 생성 및 관리
- **OCR 기능**: 스캔된 문서 텍스트 인식 (28개 언어 지원)

### 3. 학습 최적화 기능
- **학습 진도 추적**: 읽기 시간, 페이지 진행도, 복습 패턴 분석
- **지능형 학습 계획**: AI가 분석한 학습 패턴 기반 맞춤형 학습 계획
- **플래시카드 시스템**: 스마트 복습 알고리즘 기반 기억 최적화
- **협업 학습**: 그룹 주석, 실시간 문서 공유, 학습 그룹 관리
- **학습 통계**: 상세한 학습 데이터 시각화 및 분석

### 4. 사용자 경험
- **다크 모드**: 눈의 피로를 줄이는 다크 모드 및 읽기 모드
- **음성 지원**: TTS(텍스트 음성 변환) 및 음성 명령
- **멀티 디바이스 동기화**: 모든 기기에서 원활한 이용 경험
- **맞춤 설정**: 사용자화 가능한 UI, 단축키, 제스처

## 개발 환경 설정

1. Flutter 설치
```bash
git clone https://github.com/flutter/flutter.git
export PATH="$PATH:`pwd`/flutter/bin"
flutter doctor
```

2. 프로젝트 클론
```bash
git clone https://github.com/khy0425/pdf_learner.git
cd pdf_learner
```

3. 의존성 설치
```bash
flutter pub get
```

4. 환경 변수 설정
```bash
cp .env.example .env
# .env 파일을 수정하여 API 키 등 설정
```

5. 애플리케이션 실행
```bash
flutter run
```
