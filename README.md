# AI PDF 학습 도우미 (AI PDF Study Assistant)

PDF 문서를 AI를 활용하여 효과적으로 학습할 수 있도록 도와주는 Flutter 애플리케이션입니다.

## 주요 기능

### 📚 기본 기능
- PDF 파일 업로드 및 뷰어
- PDF 텍스트 추출
- 문서 관리 및 저장

### 🤖 AI 기능
- 문서 내용 자동 요약
- 핵심 개념 추출
- 맞춤형 퀴즈 생성
- Q&A 시스템

### 📝 학습 보조 기능
- 중요 내용 하이라이트
- 학습 진도 관리
- 복습 알림

## 시작하기

### 필수 요구사항
- Flutter SDK (3.0.0 이상)
- Dart SDK (2.17.0 이상)
- Android Studio 또는 VS Code
